# mwild.me

## Build / Run Locally
#### Prerequisites
* NodeJS
* NPM

#### Build
```sh
$ npm install
$ node ./index.js
```

#### Run
```sh
$ npm install
$ node ./index.js --watch
```

## Deployment

~~See the contents of `.travis.yml` for more info.~~

The site can be manually deployed by force-pushing the output of build/ to master.
